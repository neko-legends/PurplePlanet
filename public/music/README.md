# Music Assets

Bundled music for the wallpaper or screensaver lives here. Files in `public/music/` are copied into the static wallpaper build and the Lively package.

Current track:

- `The Purple Planet.mp3`

Checklist for bundled tracks:

- Generated while your Suno Pro or Premier subscription was active.
- Not generated from a free-plan song, remix, cover, copyrighted upload, third-party lyrics, or a prompt imitating a living artist.
- Exported as an `.mp3` with a clear filename.
- Listed in `ASSET-LICENSE.md` with the license granted to repo users.

The source code can remain under an open-source software license while music is licensed separately.
